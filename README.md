### Учебная практика (ознакомительная). Вариант 11

**Разработано:** Кандилян А. А., группа КИ25-17/2Б

Программа для загрузки, обработки и отображения изображений с использованием Python, OpenCV и Tkinter с дополнительными функциями. В программе реализована обработка ошибок и валидация.

---

## Описание проекта

Это десктоп-приложение позволяет пользователю:

* Загружать изображения популярных форматов с диска
* Делать снимки с веб-камеры в реальном времени
* Отображать отдельные цветовые каналы (R, G, B) от оригинального изображения
* Выполнять операции обработки согласно **Варианту 11**:
  * **Обрезка изображения:** по введенным пользователем координатам
  * **Повысить яркость:** на заданное пользователем значение
  * **Нарисовать круг:** красный круг по заданным координатам центра и радиусу
* Сбрасывать изменения до исходного состояния в любой момент

При ошибках приложение выводит подробные сообщения и рекомендации.

Использование приложения
*После запуска откроется графический интерфейс со встроенной областью просмотра.

* Используйте кнопку «Загрузить изображение» или «Сделать снимок с веб-камеры», чтобы добавить картинку в программу.

* Кнопки «Обрезка изображения», «Повысить яркость» и «Нарисовать круг» выполняют обработку по требованиям Варианта 11. * Параметры обработки (координаты, радиус, уровень яркости) вводятся пользователем в появляющихся диалоговых окнах.
* Кнопка «Сбросить» в любой момент возвращает изображение к исходному состоянию.
