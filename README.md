Учебная практика (ознакомительная). Вариант 11

Разработано: Кандиляном А. А. , группа КИ25-17/2Б

Программа для загрузки, обработки и отображения изображений с использованием Python, OpenCV и Tkinter с дополнительными функциями. В программе реализована обработка ошибок и валидация.

Требования к системе:
1) Установленный интерпретатор Python 3.6.8 .
2) Менеджер пакетов pip (поставляется вместе с Python).

Пошаговый запуск
1. Клонирование репозитория или переход в папку
Откройте терминал (или командную строку) и перейдите в корневую директорию проекта:

Bash
cd path/to/Project_sfu
(Вместо path/to/Project_sfu укажите реальный путь к папке на вашем компьютере).

2. Создание виртуального окружения
Для изоляции зависимостей проекта создайте локальное виртуальное окружение venv:

Для Windows:

Bash
python -m venv venv, (лучше "py -3.6.8 -m venv venv")
Для macOS / Linux:

Bash
python3 -m venv venv

3. Активация виртуального окружения
Активируйте созданное окружение перед установкой библиотек:

Для Windows (Command Prompt / Git Bash):

Bash
source venv/Scripts/activate или 
Для macOS / Linux (Terminal):

Bash
source venv/bin/activate
После активации в начале строки терминала появится префикс (venv).

4. Установка зависимостей
Установите все необходимые для работы приложения библиотеки одной командой из файла зависимостей:

Bash
pip install -r requirements.txt

5. Запуск приложения
Запустите главный скрипт программы:

Bash
python main.py или нажмите на флажок старта

Использование приложения
После запуска откроется графический интерфейс со встроенной областью просмотра.

Используйте кнопку «Загрузить изображение» или «Сделать снимок с веб-камеры», чтобы добавить картинку в программу.

Кнопки «Обрезка изображения», «Повысить яркость» и «Нарисовать круг» выполняют обработку по требованиям Варианта 11. Параметры обработки (координаты, радиус, уровень яркости) вводятся пользователем в появляющихся диалоговых окнах.

Кнопка «Сбросить» в любой момент возвращает изображение к исходному состоянию.

При ошибках приложение выводит подробные сообщения и рекомендации.
